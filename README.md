# RunOnSentence

使用说明

1.  Prepared

注册GitHub并下载GitHub Desktop

使用GitHub账号登录GitHub Desktop

2.  Download source file

加入项目后，进入项目网址：<https://github.com/ShimizuNana/RunOnSentence>

点击绿色按钮「Code」 → 选择「Open with GitHub Desktop」

GitHub Desktop 会自动打开并弹出窗口：

<img width="865" height="862" alt="image" src="https://github.com/user-attachments/assets/4ae7a64f-a526-4597-9caa-adb6bcd27eb1" />


选择本地保存路径（项目会被克隆到该文件夹）。点击
Clone。克隆完成后，项目文件会出现在你选定的文件夹中。

3.  Upload files

在本地修改项目内容后

打开 GitHub Desktop，软件会自动检测到更改。

左侧会显示：修改的文件名、修改内容预览

<img width="865" height="595" alt="image" src="https://github.com/user-attachments/assets/b3f325e6-0722-479e-bd76-2c824ac9deb4" />


填写提交说明 (Commit message)

在下方输入框填写简短描述

<img width="865" height="595" alt="image" src="https://github.com/user-attachments/assets/56e918e2-8001-429b-a271-3c7b4206400e" />


点击「Commit to master」。

点击右上角按钮将修改推送至远程仓库：

<img width="865" height="595" alt="image" src="https://github.com/user-attachments/assets/386f7200-1cf2-4956-8e0b-15730e1a0eff" />

4.  Fetch new changes

打开 GitHub Desktop。点击右上角的「Fetch
origin」按钮。如果有新内容，会提示「Pull origin」。

<img width="865" height="595" alt="image" src="https://github.com/user-attachments/assets/cdf44fcb-f0b6-463e-9d47-0cf349ee3664" />


点击「Pull origin」，即可把团队的最新修改合并到本地。
